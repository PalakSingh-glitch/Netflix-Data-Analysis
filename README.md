# 📊 Netflix Data Analysis Project

## 👩‍💻 About the Project
This project analyzes Netflix movies and TV shows dataset to understand content trends, ratings, and category distribution using Excel.

---

## 🎯 Objectives
- Analyze Netflix content by type (Movies vs TV Shows)
- Find most popular genres
- Study ratings distribution
- Identify content trends over years

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Data Cleaning Techniques

---

## 📈 Key Insights
- Most content on Netflix is Movies
- TV-MA and TV-14 are most common ratings
- Drama & Comedy are top genres
- Content increased significantly after 2015
